Bus Ticket Booking Console Application

This project is a **console-based Bus Ticket Booking System** developed using Java and Spring Framework in Spring Tool Suite. It simulates a real-world bus reservation system and focuses on backend logic, object-oriented programming, and modular application design.

Key Features:

  • User can view available buses and schedules
  • Seat availability checking
  • Ticket booking and cancellation
  • Fare calculation based on selected route
  • Input validation and exception handling

Technologies Used:

  • Java
  • Spring Framework
  • Spring Tool Suite (STS)
  • OOP Concepts
